# Authentication Backend

A backend authentication system built with Node.js, Express, and MongoDB.

This project includes user authentication, JWT authorization, role-based access control, and a forgot password system using Nodemailer.

## Features

- User Signup
- User Login
- JWT Authentication
- Protected Routes
- Role Based Access Control (RBAC)
- Forgot Password Functionality
- Reset Password Through Email Link
- Password Hashing using bcrypt
- MongoDB Integration with Mongoose

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Nodemailer
- dotenv
